# Blog pessoal

Esse repositório contém o código fonte do website [roveeb.com](roveeb.com), que foi construido utilizando as seguintes ferramentas:
* Hugo
* Netlify
* Github

## Instalação

```
git clone https://github.com/roveeb/site.git
```

## Uso
```
cd site/
hugo server -D
```
